# 🎬 Cinema Backend

Ce projet est le **backend** du système de réservation de tickets de cinéma, développé avec **Java + Spring Boot**.

## ⚙️ Technologies

- Java 17
- Spring Boot
- JPA / Hibernate
- MySQL
- Lombok
